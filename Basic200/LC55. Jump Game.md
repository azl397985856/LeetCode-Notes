## LC 55. Jump Game

### 思路

维护一个可以达到的最远距离
注意 while 之后出来的 i 是恰好不满足的条件的

### 代码

```JavaScript
var canJump = function(nums) {
  let i = 0
  let furthest = nums[i];
  while(i <= furthest){
    furthest = Math.max(furthest, nums[i] + i)
    i++;
  }
  // console.log(i)
  return i - 1 >= nums.length - 1
};

```

### 复杂度分析

时间复杂度：O(N)
空间复杂度：O(1)
