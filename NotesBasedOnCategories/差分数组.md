# 差分数组
和前缀和思想很相似，也是前缀处理的一种。  
差分数组的主要适用场景是**频繁对原始数组的某个区间的元素进行增减**。

## 定义
diff[i] = nums[i] - (nums[i - 1] || 0)
diff[0] = nums[0]

```JavaScript
for (let i = 1; i < nums.length; i++) {
    diff[i] = nums[i] - nums[i - 1];
}
```

## 应用

### 反推原始数组nums
```
nums[i] = nums[i - 1] + diff[i]
```
