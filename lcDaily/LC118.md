## LC 118

https://leetcode-cn.com/problems/pascals-triangle/

### 思路

遍历

### 代码

```JavaScript
var generate = function(numRows) {
  let result = [];
  for(i = 0; i < numRows; i++){
    let temp = [];
    for(j = 0; j <= i; j++){
      if(j == 0 || j == i){
        temp.push(1)
      }else{
        let last = result[result.length - 1];
        temp.push(last[j] + last[j - 1])
      }
    }
    result.push(temp)
  }

  return result;
};

```

### 复杂度分析

时间复杂度：O(N^2) ?
空间复杂度：O(N^2) ?
